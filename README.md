# IHD FORUM 20th Anniversary Edition

### Use Express/Sequelize backed blogging REST API

### API Endpoints

**GET /**:

* retrieve all messages

**POST /login**

* log in a user

**POST /message**

* create a new message

**POST /comment/:id**

* add a comment to message (id)

**POST /like/:id**

* 'like' the message as the current user
